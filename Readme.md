## Required Software

- MySQL

## Required Packages

- mysql-connector-python
- dotenv

## Run Locally

Clone the project

```bash
  git clone
```

Go to the project directory

```bash
  cd car-insurance-system
```

Execute SQL commands in ./db/car_insurance.sql file

Go to app directory

```bash
  cd app
```

Install dependencies

```bash
  pip3 install -r requirements.txt
```

Run App

```bash
  python3 app.py
```
